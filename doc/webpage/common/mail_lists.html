Mail Lists

maintains a number of email lists that users and developers can subscribe to. We recommend subscribing to the following mailing lists as appropriate.
  • -Announce Mailing List - Low-volume, primarily for release announcements. (Highly Recommended)

  • -Users Mailing List - Discussion forum for users of . (Recommended)

  • -Developers Mailing List - Discussion forum for developers. (Developers)

  • -Checkins Mailing List - CVS Commit Message. (Active Developers)

  • -Regression Mailing List - Captures output from automated regression tests. (Active Developers)
