What is Albany?

Albany is a prototype PDE code that makes heavy use of Trilinos libraries. The code serves as a PDE research platform, a template for starting new Trilinos-based application codes, and as a testbed for new algorithms. Albany is leading the way in embedded algorithms. It makes heavy use of Sacado, Stokhos, and Phalanx packages, in the PDE fill, and Dakota, Moocho, LOCA, Rythmos, and Piro to drive the solution. Top

Is Albany available?

Albany is Publically Available software. However, we have not made it generally downloadable from the web at this point. For Sandians, it can be pulled from a git repository: software.sandia.gov:/space/git/Albany. Top
