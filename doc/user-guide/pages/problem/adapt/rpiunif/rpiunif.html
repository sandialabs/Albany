Adaptation - RPI Unif Size

Description  Dependencies  Parameters  Example 

Description

At a given iteration step, MeshAdapt will uniformly remesh the problem domain based on a given isotropic mesh size.

Dependencies

SCOREC

Paramters

  • Max Number of Mesh Adapt Iterations
    • Data Type
      •   <int>
    • Default Value
      •   1
    • Description
      •   "Number of iterations to limit meshadapt to"
  • Remesh Step Number
    • Data Type
      •   <int>
    • Default Value
      •   1
    • Description
      •   "Iteration step at which to remesh the problem"
  • Target Element Size
    • Data Type
      •   <double>
    • Default Value
      •   0.1
    • Description
      •   "Seek this element size when isotropically adapting"
